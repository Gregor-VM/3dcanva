### Built with

- Canva API
- Vite
- Typescript
- pnpm

So, to run/build the project you first need to have installed [pnpm](https://pnpm.io/es/installation)

To run the project use:

```
pnpm run dev
```

To build the project run:

```
pnpm run build
```
